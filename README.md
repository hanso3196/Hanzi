# Hanzi Karteikarten PWA

Dies ist die installierbare PWA-Version der Hanzi-Karteikarten-App.

## Enthalten

- `index.html`
- `manifest.json`
- `service-worker.js`
- App-Icons in `icons/`

## Installation auf dem Handy

1. Lade den gesamten Ordner auf einen Webspace hoch.
   Geeignet sind z. B. GitHub Pages, Netlify, Vercel oder ein normaler Webserver.
2. Öffne die URL auf dem Handy.
3. Android/Chrome: Menü öffnen und „App installieren“ oder „Zum Startbildschirm hinzufügen“ wählen.
4. iPhone/Safari: Teilen-Menü öffnen und „Zum Home-Bildschirm“ wählen.

## Wichtig

Eine PWA muss normalerweise über HTTPS laufen, damit Installation und Offline-Funktion zuverlässig funktionieren.
GitHub Pages, Netlify und Vercel liefern automatisch HTTPS.

## Lokale Speicherung

Lernstände, Pakete und Fehlerzähler werden lokal im Browser gespeichert.
Nutze Export/Import in der App, um Backups zu erstellen.
